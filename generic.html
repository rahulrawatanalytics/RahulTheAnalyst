e-Commerce Orders Dashboard

In this project, a comprehensive Looker Studio dashboard was created to analyze order data. The dashboard features key metrics such as the total number of orders, customers, states, and segments. It includes visualizations for segment performance, category breakdown, shipping mode distribution, and top states by order volume. Additionally, it provides trend analysis by month and year and lists high-quantity orders. Interactive filters allow users to drill down into the data by region, segment, category, and ship mode, enabling detailed insights and data-driven decision-making.
